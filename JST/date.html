JavaScript Dates

The Date object lets you work with dates (years, months, days, hours, minutes, seconds, and milliseconds):


A JavaScript date can be written as a string: Tue May 16 2017 16:26:36 GMT+0530 (India Standard Time)
or as a number: 1494932196407
Dates written as numbers, specifies the number of milliseconds since January 1, 1970, 00:00:00..

Example


<!DOCTYPE html>
<html>
<body>
<h2>JavaScript DATE</h2>
<p></p>
<p id="demo"></p>
<script>
document.getElementById("demo").innerHTML = Date(); </script>
</body>
</html>
