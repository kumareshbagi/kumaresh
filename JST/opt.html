JavaScript Object Prototypes

Every JavaScript object has a prototype. The prototype is also an object.


All JavaScript objects inherit their properties and methods from their prototype:

Example


<!DOCTYPE html>
<html>
<body>
<h2></h2>
<p></p>
<p id="demo"></p>
<script>
function Person(first, last, age, eye) { this.firstName = first; this.lastName = last; this.age = age; this.eyeColor = eye; } var myFather = new Person("John", "Doe", 50, "blue"); var myMother = new Person("Sally", "Rally", 48, "green"); document.getElementById("demo").innerHTML = "My father is " + myFather.age + ". My mother is " + myMother.age; </script>
</body>
</html>
