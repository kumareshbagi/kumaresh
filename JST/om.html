JavaScript Object Methods

JavaScript methods are the actions that can be performed on objects.


A JavaScript method is a property containing a function definition.:

Example


<!DOCTYPE html>
<html>
<body>
<h2>Creating and using an object method.:</h2>
<p>A method is actually a function definition stored as a property value.</p>
<p id="demo"></p>
<script>
var person = { firstName: "John", lastName : "Doe", id : 5566, fullName : function() { return this.firstName + " " + this.lastName; } }; document.getElementById("demo").innerHTML = person.fullName(); </script>
</body>
</html>
