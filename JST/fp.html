JavaScript Function Parameters

A JavaScript function does not perform any checking on parameter values (arguments).


Function parameters are the names listed in the function definition.:

Example


<!DOCTYPE html>
<html>
<body>
<h2>Setting a default value to a function parameter.</h2>
<p></p>
<p id="demo"></p>
<script>
function myFunction(x, y) { if (y === undefined) { y = 0; } return x * y; } document.getElementById("demo").innerHTML = myFunction(4); </script>
</body>
</html>
