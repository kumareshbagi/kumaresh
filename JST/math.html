JavaScript Math Object

The JavaScript Math object allows you to perform mathematical tasks on numbers.


Math.round() Math.round(x) returns the value of x rounded to its nearest integer:. Math.pow() Math.pow(x, y) returns the value of x to the power of y: Math.sqrt() Math.sqrt(x) returns the square root of x:
