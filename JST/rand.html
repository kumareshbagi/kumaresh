JavaScript Random

Math.random() Math.random() returns a random number between 0 (inclusive), and 1 (exclusive):


JavaScript Random Integers Math.random() used with Math.floor() can be used to return random integers.

Example


<!DOCTYPE html>
<html>
<body>
<h2>JavaScript Random</h2>
<p>Math.floor(Math.random() * 100) + 1) returns a random integer between 1 and 100 (both included):</p>
<p id="demo"></p>
<script>
document.getElementById("demo").innerHTML = Math.floor(Math.random() * 100) + 1; </script>
</body>
</html>
