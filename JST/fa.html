JavaScript Validation API

checkValidity() Returns true if an input element contains valid data.


setCustomValidity() Sets the validationMessage property of an input element:

Example


<!DOCTYPE html>
<html>
<body>
<h2>JavaScript Validation API</h2>
<p>Enter a number and click OK:</p>
<p id="demo">If the number is less than 100 or greater than 300, an error message will be displayed.</p>
<script>
function myFunction() { var inpObj = document.getElementById("id1"); if (inpObj.checkValidity() == false) { document.getElementById("demo").innerHTML = inpObj.validationMessage; } else { document.getElementById("demo").innerHTML = "Input OK"; } } </script>
</body>
</html>
