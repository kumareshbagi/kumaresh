JavaScript Closures

JavaScript variables can belong to the local or global scope.
Global variables can be made local (private) with closures.


A function can access all variables defined inside the function, like this:

Example


<!DOCTYPE html>
<html>
<body>
<h2>A function can access variables defined inside the function:</h2>
<p></p>
<p id="demo"></p>
<script>
function myFunction() { var a = 4; document.getElementById("demo").innerHTML = a * a; } </script>
</body>
</html>
